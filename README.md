# Automobile Insurance Analytics

## Project Overview
This project focuses on analyzing automobile insurance data to identify trends in customer complaints, premium amounts, issue categories, resolution time, and provider performance. The goal is to uncover key insights that help insurance companies improve service quality and customer satisfaction.

## Tools Used
- Excel
- Power BI (optional if you used)

## Analysis Performed
- Complaint analysis by insurance provider  
- Premium trend analysis across years  
- Identification of major complaint categories  
- Resolution time comparison across providers  
- Customer issue frequency analysis  
- Insights into high-risk complaint regions or product types

## Key Insights
- Certain complaint categories have consistently higher frequency  
- Specific providers show longer resolution times  
- Premium amounts vary significantly across customer segments  
- Complaint volume rises during specific periods or policy types  

## Business Impact
This analysis helps insurance companies:
- Improve customer satisfaction  
- Reduce complaint resolution time  
- Identify and fix high-risk areas  
- Optimize premium pricing strategies
